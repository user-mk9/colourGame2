About







This plugin (darkmode.js) is very good anad easy to use, it saves a developer from having to actaully write the code in order to perform a darkmode feature. This is just a shortcut use darkmode.

For this application there is widget which I toggled on

Saving user choice

CSS can be applied to ths plugin:

  • Designing the widget to activate and/or deactivate darkmode
  • Designing the look of the widget
  • "darkmode--activated" can be used to switch or change specific elements when darkmode is activated
  • "darkmode-ignore" can be used to where you don't want to apply darkmode, you can also use "isolation: isolate;" to do the same
  • "mix-blend-mode: difference;" is useed to revert the dark-mode with this style
